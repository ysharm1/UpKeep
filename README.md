# UpKeep

A platform connecting homeowners with local service providers for HVAC, plumbing, and home repair issues.

## Features

- AI-powered diagnostic chatbot with video/photo upload
- Guided troubleshooting for common home issues
- Direct connection to local service providers
- Simple booking and contact management

## Tech Stack

- Frontend: React/Next.js
- Backend: Node.js
- Database: PostgreSQL
- AI: OpenAI API
- File Storage: Cloud storage for media uploads
